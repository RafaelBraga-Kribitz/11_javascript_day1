EX00 - Basic Exercises

  • Exercise 3: Array of Players
    • Create an array called players with the names: Martin, Thomas, Peter, Mathias, and Niki.
    • Output the third player in the console as: "The most valuable player of the match is [Third Player Name]".

Solution

To solve this exercise, we need to create an array called players with the names: Martin, Thomas, Peter, Mathias, and Niki. We can use the <script> tag to include JavaScript code.

Code

      <script>
        var players = ["Martin", "Thomas", "Peter", "Mathias", "Niki"];

        console.log("The most valuable player of the match is " + players[2]); // Peter
        console.log("The least valuable player of the match is " + players[0]); // Martin
      </script>

Result

The result of this exercise is to print "The most valuable player of the match is Peter" and "The least valuable player of the match is Martin" in the browser console.
