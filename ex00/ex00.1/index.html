EX00 - Basic Exercises

  • Exercise 1: Internal JavaScript in HTML
    • Print "Hello from the Console" in the browser console.
    • Display "Hello from the Browser" on the web page.

Solution

To solve this exercise, we need to add the JavaScript code directly within the HTML file. We can use the <script> tag to include JavaScript code.

Code

      <script>
        console.log("Hello from the Console");
      </script>

Result

The result of this exercise is to print "Hello from the Console" in the browser console.
