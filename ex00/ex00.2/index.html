• Exercise 2: External JavaScript File
    • Create a variable name and assign your name.
    • Update the console log to display "Hi, my name is [Your Name]".
    • Create a variable age and set it to your age.
    • Update the console log to display "Hi, my name is [Your Name] and I am [Your Age] years old."

Solution

To solve this exercise, we need to create a JavaScript file called script.js and include it in the HTML file. We can use the <script> tag to include JavaScript code.

Code

      <script>
      var userName = "Rafael";
      var age = 42;
      
      console.log("Hi, my name is " + userName + " and I am " + age + " years old.");      
      </script>

Result

The result of this exercise is to print "Hi, my name is Rafael and I am 42 years old." in the browser console.
